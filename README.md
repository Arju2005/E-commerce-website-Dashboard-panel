# E-commerce Website Dashboard Panel 🛒📊

A modern and responsive E-commerce Dashboard Panel built using HTML, CSS, and JavaScript.  
This dashboard helps manage products, orders, customers, sales analytics, and other e-commerce activities through an interactive admin panel interface.

## 🚀 Features

- Responsive dashboard layout
- Sidebar navigation menu
- Product management section
- Orders and customer details
- Sales analytics cards
- Interactive charts and statistics
- Clean and modern UI design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📂 Project Structure

```bash
E-commerce-Website-Dashboard-panel/
│── index.html
│── style.css
│── script.js
│── assets/
└── README.md
```

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/E-commerce-Website-Dashboard-panel.git
```

2. Open the project folder

```bash
cd E-commerce-Website-Dashboard-panel
```

3. Open `index.html` in your browser.
